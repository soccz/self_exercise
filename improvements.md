# Iron Quant 2.0: 전략 로드맵 (사용자 중심)

## 🎯 현재 상태: "안정적인 MVP"
성공적으로 기능적인 MVP를 배포했습니다.
- **완료**: DB 스키마, API, 텔레그램 웹훅, 자동 리마인더(Cron).
- **상태**: **기술적 부채 제로.** 마지막 SQL 패치만 잘 적용되었다면 시스템은 100% 준비되었습니다. 이제 "사용"만 하면 됩니다.

## 🚀 다음 단계: "공격적 성장 전략"
*최고 수준의 피트니스 & 생산성 앱(Strong, Hevy, Streak 등)을 분석하여 도출한 앙상블 전략입니다.*

### 1. 무마찰 데이터 입력 (Zero-Click Protocol)
**문제점**: 운동 중에 `스쿼트 100 5 5`라고 타자를 치는 건 흐름을 끊습니다. 귀찮아서 기록을 안 하게 됩니다.
**해결책**:
- **보이스 로깅 (Whisper API)**: 텔레그램 녹음 버튼을 누르고 말하세요. *"스쿼트 100키로 5회 5세트 완료."* -> 봇이 알아서 텍스트로 변환하고 기록합니다. (가장 추천)
- **포토 로깅 (GPT-4o Vision)**: 애플워치 요약 화면이나 헬스장 머신 계기판을 찍어서 보내면 -> 숫자를 자동 인식합니다.

### 2. 시각적 "자산" 리포트 (Bloomberg Terminal Experience)
**문제점**: 텍스트로만 "볼륨 10,000kg"라고 하면 감흥이 없습니다(도파민 부족).
**해결책**:
- **차트 이미지 전송**: 텍스트 대신, 지난 3개월간의 벤치프레스 1RM 추세선을 **`.png` 이미지 차트**로 생성해서 보내줍니다.
- **"주간 배당금" 리포트**: 매주 일요일, 시각화된 요약본을 받습니다.
    - *매수/매도 시그널*: 볼륨이 늘었으면 초록색 상향 화살표(Bull), 줄었으면 빨간색(Bear).
    - *배당금 지급*: "이번 주 300 XP (근성장) 배당금을 획득했습니다."

### 3. 스마트 코칭 (Algo-Trading Engine)
**문제점**: 지금 봇은 단순히 받아적기만 하는 "서기(Loggger)"입니다. 관리해주는 "매니저(Manager)"가 되어야 합니다.
**해결책**:
- **장전 브리핑 (Pre-Market)**: 헬스장 가기 전(예: 오후 6시)에 봇이 먼저 말을 겁니다. *"지난번 가슴 운동 때 100kg 실패하셨죠? 오늘은 95kg로 8회 해서 모멘텀을 확보하세요. 지정가 매수 추천."*
- **RPE 기반 자동 조절**: 만약 RPE 10(죽을 만큼 힘들다)을 기록하면, 봇이 즉시 개입합니다. *"시장 과열 상태. 부상 위험 감지. 다음 세트는 무게를 10% 낮추세요."*

### 4. 비교와 경쟁 (Market Index)
**문제점**: 혼자 기록하면 외롭고 기준이 모호합니다.
**해결책**:
- **"시장 평균" 비교**: 내 `스쿼트/체중` 비율을 일반적인 스트렝스 기준과 비교해줍니다. (예: "당신은 S&P 500 기준 상위 15%입니다.")
- **고스트 리플레이**: "작년 오늘, 당신은 80kg를 들었습니다. 오늘은 100kg네요. (YoY +25% 성장)."

## 🛠 즉시 실행 가능한 추천 항목 (Low Effort, High Impact)
1.  **차트 통합 (`quickchart.io`)**: 텔레그램에 진짜 '차트 이미지'를 보내는 기능. (구현 쉽고 만족도 최상)
2.  **보이스 노트 기능**: 텔레그램 `audio` 타입을 받아서 텍스트로 변환만 하면 끝입니다. (사용성 혁명)
